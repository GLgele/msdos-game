# MSDOS-GAME

#### 介绍
A MSDOS Game

#### 软件架构
head.h
   |
cpp.cpp
   |
main.cpp

#### 安装教程

1.  下载本代码仓库（.zip/https/svn/release均可）
2.  找到main.exe
3.  运行

#### 使用说明

1.  按提示操作即可
2.  退出时请在主界面退出，不要直接关闭，否则会导致游戏进度丢失！
